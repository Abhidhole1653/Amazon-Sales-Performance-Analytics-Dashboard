# Amazon Sales Performance Analytics Dashboard

## üìä Project Overview

This project presents an **end-to-end Amazon Sales Analytics solution** built using **Power BI** with a strong focus on **business insights, performance tracking, and decision-making metrics**. The dashboard analyzes **sales, revenue, profit, customers, orders, and targets** across multiple dimensions such as **year, category, country, state, weekday/weekend, and order status**.

To demonstrate **data analytics versatility**, this repository also includes **SQL queries** used for exploratory analysis and validation of KPIs prior to dashboard creation.
